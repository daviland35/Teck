Kasier :

 

Les utilisateurs peuvent récupérer leurs commandes de matériels informatique via les Kasiers.

Il s’agit de casiers connectés 

Lorsque l’utilisateur devra récupérer une commande, il recevra un

Mail contenant :

-              Le numéro de commande.

-              Le numéro de casier et le lieu.

-              Le mode opératoire pour récupérer la commande.

-              Un QR Code et un code à 6 chiffres pour ouvrir le Kasier.

 

L’article  KB0025680 détaille le mode opératoire pour récupérer une commande.

 

En cas d’incident ou l’utilisateur ne se souvient pas du code de retrait ou qu’il ne fonctionne pas, on peut renvoyez en code (KB0025681) après avoir vérifié son identité
